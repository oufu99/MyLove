第一节  可用的快捷键组合

Ctrl/Alt/Shift/Win（任意按键及其组合）与 F1～F12、A～Z、0～9、Insert、Del、Home、
End、PgUp、PgDn、Space、BackSpace、Tab、+ - * /、Enter、Esc、方向键 的组合。

自定义快捷键，应注意避免与系统快捷键冲突。

更新说明：

2010年9月10日  为避免与菜单项快捷键冲突，尽量减少「Alt+字母」形式的自定义快捷键，
只保留了 "Alt + A"，用于设置文件日期/属性。

2017年12月8日  修改 "Ctrl+J"、"Ctrl+K" 及 "Ctrl+Y" 快捷键，用于访问「桌面」、「我
的电脑」及「锁定/解锁当前标签（允许更改文件夹）」 。

2017年12月16日  修改部分快捷键：Alt+Shift+0 显示/隐藏：垂直工具栏、Alt+Shift+F6 
测试压缩文件、Ctrl+5 拆分文件、Ctrl+6 合并文件、Ctrl+7 编码文件、Ctrl+8 解码文件、
Ctrl+Shift+9 内部关联、Ctrl+Shift+F7 反向排序、Ctrl+Shift+F 搜索文件（单独进程）、
Shift+F7 编辑当前路径、Shift+F9 断开 FTP 连接。

列表说明：
+、-、*、/，表示位于数字键盘中的对应按键。
快捷键后带有[+]，表示是本版本新添加的快捷键；
快捷键后带有[*]，表示是更改功能的快捷键；
快捷键后带有[s]，表示是出自 Shanny 的版本的快捷键。

第四节  命令行快捷键

如果 Total Commander 是活动窗口，几乎所有的按键都直接输入到命令行。

以下是一些快捷键：

Enter                           如果命令行有字符，则执行命令。（否则，启动来源窗口
                                光标处程序）。如果是内部命令，则执行该命令。否则，
                                执行指定名称的程序。
Shift + Enter		        通过 Windows 外壳执行命令行中的命令、且不关闭程序窗口。
                                仅存在 NOCLOSE.EXE 时有效！
Ctrl + Enter                    将来源窗口光标处的文件名添加到命令行尾部。
Ctrl + Shift + Enter	        将来源窗口光标处的文件名（含路径）添加到命令行尾部。
Ctrl + 下方向键 或 Alt + F8	打开命令行历史列表。按住 Ctrl 键并使用向上和向下方向键
                                选择命令。按下左右方向键后，可编辑命令行内容。
Tab			        切换到来源窗口。
左右方向键                      按下左右方向键可在命令行中移动光标。
Esc                             清除命令行，光标回到文件窗口。
Ctrl + A                        选定整个命令行
Ctrl + Y                        清除命令行，光标仍留在命令行。
Ctrl + C                        复制选定文本到剪贴板
Ctrl + X                        剪切选定文本到剪贴板
Ctrl + V                        从剪贴板粘贴文本到命令行
Ctrl + E                        复制前一命令到命令行
Ctrl + K                        删除到行尾
Ctrl + W                        删除光标左侧的单词
Ctrl + T                        删除光标右侧的单词
